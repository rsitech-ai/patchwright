# Patchwright 0.1.0 Release Notes

Patchwright 0.1.0 introduces a local-first macOS workspace for reviewing
authorized GitHub repositories, turning issues and pull requests into durable
engineering tasks, supervising Codex work, and keeping remote delivery behind
explicit approvals.

Highlights:

- repository, issue, pull-request, review, CI, and workflow visibility;
- isolated task worktrees and durable lifecycle checkpoints;
- streamed Codex task sessions with cancellation and recovery;
- exact-preview approval gates for GitHub delivery and merge;
- GitHub App webhook ingestion and scoped installation-token operations;
- release provenance, privacy manifest, bounded logs, and automation kill switch;
- hardened repository-origin, review-head, and release-candidate checks;
- responsive, timeout-bounded GitHub App health verification.
